# 廟埕守香 專案文件

## 專案概述

《廟埕守香》是一款以台灣神明信仰為主題的文字塔防遊戲。玩家在廟埕棋盤上請出法器與神名字，阻止妖魔鬼怪沿路徑逼近香爐。遊戲的核心特色是「中文字」本身就是單位，並透過合成、神明組合、部首攻擊動畫與台灣廟宇視覺，形成辨識度。

目前專案是純前端靜態網頁，不需要後端服務。

## 核心玩法

玩家使用香火請令，抽出一般法器或神名字，拖曳到棋盤可放置格上鎮守。

- 目標：守住香爐，撐過 12 波妖魔。
- 資源：香火。
- 抽取：按下「請令」消耗香火。
- 放置：將下方法壇中的字拖曳到棋盤。
- 合成：相同字、相同等級可以合成升級。
- 丟棄：將字拖到「化去」圖示，或選取後按「化去」，可回收部分香火。
- 敗北：妖魔抵達香爐會扣香爐耐久，耐久歸零失敗。

## 操作方式

- 拖曳法壇中的字到棋盤：出戰。
- 拖曳棋盤上的字到其他可放置格：移動。
- 拖曳相同等級的同字：合成升級。
- 拖曳字到「化去」：丟棄並回收香火。
- 點選字後按「化去」：丟棄選取字。
- 點選「重開」：重新開始。

## 一般法器

一般法器是 `符、鏡、鈴、劍、印`。一般法器最多升到 3 級。

| 字 | 名稱 | 定位 | 攻擊方式 |
| --- | --- | --- | --- |
| 符 | 鎮煞符 | 大範圍攻擊 | 符陣掃向多個敵人，不暈、不慢 |
| 鏡 | 照妖法鏡 | 遠距多目標照射 | 鏡光照妖，照射多個目標 |
| 鈴 | 淨壇法鈴 | 緩速牽制 | 鈴聲震盪，使妖魔短時間變慢 |
| 劍 | 斬煞法劍 | 近距高傷 | 對單體造成高傷，附斬擊效果 |
| 印 | 敕令法印 | 鎮壓控制 | 對單體造成傷害並暈眩 |

## 神名字與組合

神名字會隨機請出，不由一般法器升級而來。神名字單獨放置時不啟動攻擊，必須與對應另一字相鄰才會成為神明組合。

| 組合 | 神明 | 定位 | 攻擊方式 |
| --- | --- | --- | --- |
| 媽 + 祖 | 媽祖 | 遠距穿透 | 對多個目標穿透攻擊 |
| 關 + 公 | 關公 | 近距斬擊 | 對範圍內目標斬擊 |
| 城 + 隍 | 城隍爺 | 鎮壓控制 | 傷害並暈眩敵人 |
| 觀 + 音 | 觀音 | 遠距穿透 | 對多個目標穿透攻擊 |
| 王 + 爺 | 王爺 | 巡擊 | 對多個目標巡行攻擊 |

神明組合規則：

- 兩字相鄰才會啟動。
- 啟動後才可以繼續升級。
- 神明組合字不再限制 3 級，可以一直升上去。
- `媽` 與 `祖` 本身沒有攻擊力，組成 `媽祖` 後才會攻擊與升級。
- `觀` 只能隨機出現，不會由其他字升級而來。

## 敵人

敵人以妖魔鬼怪字形呈現，不使用黑色圓圈。

| 字 | 名稱 | 特性 |
| --- | --- | --- |
| 怪 | 小怪 | 基礎敵人 |
| 鬼 | 野鬼 | 較耐打、較慢 |
| 妖 | 山妖 | 移動較快 |
| 魔 | 魔王 | 高血量、抵達香爐扣較多耐久 |

敵人會有小型血條，血條位於字上方。被鈴影響時，走路動畫變慢；被印或城隍爺鎮壓時，才會暈眩。

## 升級與化去

### 合成升級

- 一般法器：同字、同等級可合成，最高 3 級。
- 神名字：同字、同等級可合成，但必須位於已啟動神明組合的格子上。
- 神名字啟動後可持續升級，沒有固定上限。

### 化去回收

化去會刪除字並回收香火。

- 一般法器回收較少。
- 神名字回收較多。
- 等級越高，回收越多。
- 棋盤上的字化去時，會在原位置顯示 `+香火`。

## 美術與介面方向

目前視覺方向是台灣廟埕、廟宇、法壇、朱印、香火。

已使用圖片素材：

- `assets/temple-backdrop.webp`：廟埕背景。
- `assets/temple-board.webp`：棋盤與面板質感。
- `assets/discard-burner.webp`：化去圖示。

設計注意事項：

- 字要清楚，不使用會讓字糊掉的光暈。
- 棋盤不震動，避免暈眩。
- 出戰字下方不加額外色條。
- 出戰的特殊字不加白色方塊底。
- 攻擊時才做部首拆動，平常不拆小字。
- 法壇中的字不要動。
- 連結框應像朱印，不要像 debug 紅框。
